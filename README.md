# CIMIS Backend API (Mobile Support)

CIMIS Backend is a Node.js + Express REST API built to support the **CIMIS Mobile Interface** (Ionic React App).

It provides endpoints for:

- Master data (Regions, Districts, TAs, Village Clusters)
- Beneficiaries management
- Verified beneficiaries (global + per device)
- Bulk syncing group assignments from mobile devices
- Dashboard counts (total verified + verified by device)

---

## 🚀 Features

✅ Mobile-friendly API responses  
✅ Supports offline-first mobile sync workflows  
✅ Device-based verification tracking  
✅ Bulk sync support for group assignment  
✅ Simple REST API structure  
✅ Easy deployment (Render / Railway / VPS)

---

## 🛠 Tech Stack

- Node.js
- Express.js
- REST API
- Morgan (request logging)
- CORS
- dotenv (environment variables)
- Database (MySQL / PostgreSQL / MSSQL depending on your setup)
